How to use the tool

Steps to select the right SOC Code for you

  • Enter your 'Job Description'
  • Click on the 'Find SOC' button to get the most probable 'SOC' codes according to the tool.
  • In order to save the search, please select the correct 'SOC' code from the displayed options and click on the 'Submit' button.
  • If you do not find any appropriate options, please enter an 'SOC' code manually according to your expert opinion and click on the 'Submit' button.
*Note: Once the SOC Code is submitted, you will not be able to alter the data. It will be stored into the database directly.
About the tool

The SOC Finder Tool is developed as part of Knowledge Transfer Partnership (KTP) Project between the University of Bradford and A Y & J Solicitors.
This two-year pilot project has been launched to develop an ‘Explainable Artificial Intelligence’ (AI) system to deal with immigration applications. The pilot study will effectively automate immigration applications, taking real lawyers out of the equation for most of the process. It aims to provide huge time and cost savings.

About the tool :
Find the SOC code for an applicant based on its Job Profile with a click of a button.

For every SW visa application, one of the main tasks that a caseworker performs is going through the Job Description/Duties/Roles in the applicant’s job offer and then match it to the correct SOC code manually. We have come up with a solution that automates this task for them with the help of ‘Machine Learning’.

For a solution like this, a Machine automatically produces a result according to what it has learned from the data that we feed into it. There are several SOC codes. We have started with classifying 26 SOC Codes by teaching the Machine about these codes with hundreds of Job Description examples fetched from the company’s database and other open-source resources. The experts’ advice is also considered while creating the solution pipeline.

To further evolve our solution, we need your assistance in data collection so that we can accommodate all the codes in our tool in near future. Hence, we request you to submit the search result and help us take this forward. The submission will save the search result in our database facilitating more example to learn from for our Machine.
